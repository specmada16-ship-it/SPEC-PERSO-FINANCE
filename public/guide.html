L'application

Simple à installer.
Fonctionne partout.

Kajy est une PWA — pas besoin de passer par l'App Store. Elle s'installe depuis le navigateur et tourne hors ligne.

📱
Installable
iPhone : Safari → Partager → Sur l'écran d'accueil.
Android : Chrome → 3 points → Ajouter.
PC : icône d'installation dans la barre d'adresse.
🌐
Hors ligne
Après la première ouverture avec connexion, l'app fonctionne sans internet. Toutes les données restent accessibles.
💾
Sauvegarde régulière
Les données sont stockées localement. Exporter en JSON régulièrement — Profils → ↓ — pour ne pas tout perdre si le cache est effacé ou si tu changes d'appareil.
La méthode

Tes enveloppes,
ta réalité.

L'app propose des enveloppes par défaut — mais tu crées les tiennes selon ta vie. La seule enveloppe fixe est la Trésorerie : c'est la réserve intouchable, toujours présente.

🟡
Trésorerie — toujours là
Réserve intouchable. Ne compte jamais dans le solde disponible. Contient les Sinking Funds (épargne ciblée vers des objectifs précis).
Coussin 3 mois Sinking funds Objectifs longs terme 🔒 Non supprimable
Sinking Funds — à l'intérieur de la Trésorerie

Un Sinking Fund c'est une enveloppe d'épargne à l'intérieur de la Trésorerie avec un objectif précis et un montant cible. L'argent reste dans la Trésorerie — il est juste étiqueté.

🎯
Créer un SF
Accueil → + Nouveau Sinking Fund → nom + icône + montant cible + versement mensuel indicatif. L'app calcule automatiquement le délai estimé.
➕
Alimenter
Chaque versement est déduit du solde Trésorerie. Le bloc Trésorerie affiche en temps réel : Alloué SF et Disponible SF. Le total des SF ne peut pas dépasser le solde Trésorerie.
✅
Utiliser
Objectif atteint → bouton Utiliser ce fond. Partiel ou total. Après utilisation : choisir de clôturer (supprimer le SF) ou remettre à zéro (recommencer l'épargne).
💡 Exemple : SF "Laptop" → objectif 500 000 Ar, versement 50 000 Ar/mois → atteint en ~10 mois. Pendant ce temps, l'argent reste dans la Trésorerie et fructifie visuellement.

Dans l'app

Par où commencer.

Au premier lancement, un onboarding guidé configure tout en quelques étapes. Voici ce qui se passe.

01

Installer l'app

Safari → Partager → Sur l'écran d'accueil (iPhone). Chrome → 3 points → Ajouter (Android). Icône d'installation dans la barre d'adresse (PC). L'icône Kajy apparaît comme une vraie app.

02

Onboarding au premier lancement

L'app te guide en 7 étapes : langue → nom du profil → devise → enveloppes → sous-catégories → types de revenus. Tout est configurable ici — tu n'es pas obligé de garder les exemples proposés. Un profil importé depuis un backup .json saute ces étapes.

03

Choisir sa devise

9 devises disponibles : Ar, XOF, XAF, EUR, MUR, KMF, DJF, SCR, USD. La devise s'affiche partout dans l'app et est modifiable par profil depuis les paramètres (⚙ sur chaque profil).

04

Entrer le premier revenu

Bouton + → Revenu → montant → type de revenu. L'app affiche l'aperçu de répartition en temps réel. Valide — les enveloppes se remplissent automatiquement selon les % configurés.

05

Enregistrer chaque dépense

Bouton + → Dépense → montant → sous-catégorie. La couleur de l'enveloppe confirme que tu es au bon endroit. Si le solde est insuffisant, l'app propose de couvrir le manque depuis une autre enveloppe avant de forcer.

06

Ajouter un deuxième profil

Nom du profil en haut à gauche → bouton +. L'onboarding se relance pour le nouveau profil. Chaque profil est totalement indépendant — ses propres enveloppes, transactions, sinking funds, devise.

Sécurité

Ton code,
ton contrôle.

Chaque utilisateur crée son propre code PIN à 4 chiffres. L'app ne se verrouille pas automatiquement — c'est toi qui décides quand la verrouiller.

🔑
Créer son code
Au premier lancement, l'app demande de définir un code à 4 chiffres. Ce code est propre à chaque appareil.
🔒
Verrouiller manuellement
Profils → bouton 🔒 en bas à droite. L'app se verrouille immédiatement. Le code est demandé à la prochaine ouverture.
🔄
Changer le code
Profils → bouton 🔑. L'ancien code est demandé avant de pouvoir en définir un nouveau. Ne pas l'oublier — pas de réinitialisation sans lui.

C'est suffisant.

Trésorerie

Urgence Trésorerie.
Triple sécurité.

La Trésorerie est intouchable — sauf vraie urgence. Un bouton 🚨 discret est disponible dans le bloc Trésorerie, mais seulement si le coussin minimum est atteint.

01

Coussin de sécurité

Le bouton 🚨 Urgence est grisé tant que la Trésorerie est en dessous du coussin minimum. Par défaut : 100 000 Ar. Modifiable dans Profils → ⚙ → Coussin de sécurité.

02

Confirmation obligatoire ‼️

Un écran de confirmation s'affiche : "C'est vraiment une urgence ?" Tu dois saisir une description et un montant avant de continuer.

03

Code PIN requis

Ton code PIN personnel est demandé pour valider. Sans le bon code — pas d'accès. La dépense est enregistrée avec le badge 🚨 dans l'historique.

Sinking Funds

Utiliser un Sinking Fund.

Quand l'objectif est atteint — ou à tout moment partiellement — tu peux utiliser les fonds accumulés.

01

Bouton "Utiliser"

Chaque Sinking Fund a un bouton "Utiliser partiellement". Quand l'objectif est atteint à 100%, il devient "🎯 Utiliser ce fond" mis en évidence.

02

Choisir le montant et le label

Tu peux utiliser tout ou partie du fond. Le label de la dépense est pré-rempli avec le nom du SF — modifiable. La dépense apparaît dans l'historique avec "(SF)".

03

Clôturer ou remettre à zéro

Clôturer : le SF est supprimé après utilisation. Remettre à zéro : le SF reste actif et tu recommences l'épargne vers le même objectif.

Dépenses

Solde insuffisant.
Pas de blocage.

Si tu essaies de dépenser plus que ce que contient une enveloppe, l'app ne bloque pas — elle t'accompagne.

01

Détection automatique

Si le montant dépasse le solde de l'enveloppe, une modale s'ouvre avec le résumé : solde disponible, montant tenté, manque.

02

Choisir une enveloppe de secours

Tu choisis quelle autre enveloppe couvre le manque. L'app affiche ce que chacune peut couvrir. Les enveloppes vides sont grisées.

03

Résultat

L'enveloppe principale passe en négatif du montant total. L'enveloppe de secours est déduite du manque qu'elle couvre. Visible sur l'accueil en rouge jusqu'au renflouement.

Sauvegarde

Export et Import.

Tes données sont locales — elles restent sur ton appareil. L'export .json est le seul moyen de les sauvegarder ou de les transférer.

01

Exporter un profil

Profils (haut gauche) → icône ↓ sur le profil. Un fichier kajy-[nom]-[date-heure].json est téléchargé. Chaque export a un nom unique incluant la date et l'heure — tu peux avoir plusieurs sauvegardes du même profil sans risque d'écrasement.

02

Importer sur un nouvel appareil

Lors de la création d'un nouveau profil → écran de bienvenue → "Importer un profil existant" → choisir le fichier .json. Toutes les données sont restaurées.

03

Bonne pratique

Exporter avant de changer d'appareil, avant une mise à jour importante, ou une fois par mois comme routine. Conserve les fichiers dans un dossier cloud (iCloud, Google Drive).
